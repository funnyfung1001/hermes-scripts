## 1. 方案概述

### 1.1 目的

建立一套可落地的售后服务体系，实现以下目标：

- **保障终端客户利益** — 明确的质保承诺和报修通道
- **约束EPC安装质量** — 强制技术交底+质量免责协议
- **明确四方权责** — 品牌方、ND、EPC、客户各司其职
- **降低合规风险** — 不签协议不提供质保

### 1.2 核心原则

| 原则 | 说明 |
|------|------|
| ⚙️ 协议驱动 | 质保以协议签署为前提，不签不保 |
| 📋 流程闭环 | 保修卡随货→签署→回传→激活 |
| 🏛️ 渠道管理 | ND作为中间节点统一管理EPC |
| 🔒 风险隔离 | 品牌方对制造缺陷负责，EPC对安装质量负责 |

### 1.3 适用范围

本方案适用于尼日利亚境内所有通过授权渠道（ND Fundco）销售的TranESS C&I储能产品安装项目，包括：

- **Itel Energy产品线**：IHY-50KH3S、IC-H60X、IB-H51100、IPX-51314SPT
- **DYQUE产品线**：DC-125K241-YG1 光储一体化系统

## 4. 质保政策

### 4.1 质保期限

| 组件 | 标准质保期 | 起始日 |
|------|-----------|--------|
| IHY-50KH3S 逆变器 | 5年 | 保修卡回传激活日 |
| IC-H60X 高压箱 | 5年 | 同上 |
| IB-H51100 / IPX-51314SPT 电池包 | 5年 | 同上 |
| DC-125K241-YG1 系统 | 5年 | 同上 |
| 配件（端子/仪表/线缆/连接器） | 不在质保范围 | — |

### 4.2 质保生效条件（四项必须全部满足）

1. ✅ 设备通过合规渠道（ND Fundco）购买
2. ✅ 品牌方C&I技术团队参与安装前方案审核
3. ✅ EPC与客户签署技术交底协议和质量安全免责协议（ND见证）
4. ✅ 保修卡副本（含三方签字）回传品牌方

> ⚠️ **未完成以上任何一项 → 品牌方不提供质保**

### 4.3 质保范围

**涵盖：** 制造缺陷、材料缺陷、工艺缺陷、出厂检验未覆盖的隐藏缺陷

**免责（以下情况不保）：**
- 安装不当或不按规范操作
- 人为损坏、滥用、误操作
- 未经授权的改装或维修
- 不可抗力（雷电、洪水、火灾、地震等）
- 正常老化、磨损
- 未签署技术交底或质量免责协议的
- 未按使用环境要求运行的（温度/湿度/海拔超标）
- 连续6个月未运行或未充电
- 安装时未提前通知C&I技术团队的

## 5. 各方操作指南

### 5.1 品牌方操作指南

**项目前：**
- [ ] 确认销售渠道为授权ND（Fundco）
- [ ] 向ND提供保修卡（随货发出）
- [ ] 在系统中登记SN码（状态：PENDING）
- [ ] 审核EPC资质（培训记录、电工执照）

**安装中：**
- [ ] 审核安装方案和BOM
- [ ] 远程/现场参与调试
- [ ] 指导ND完成文件签署见证

**安装后：**
- [ ] 收到保修卡副本 → 审核签字完整性
- [ ] 审核通过 → 在系统中激活SN码（状态：ACTIVE）
- [ ] 存档保修卡副本（保存期限 ≥ 质保期+2年）

### 5.2 ND（国包）操作指南

**签约阶段：**
- [ ] 与EPC签署年度MOU框架协议
- [ ] 获取EPC的营业执照、资质证书、电工执照复印件
- [ ] 向EPC说明文件要求和技术标准

**项目实施：**
- [ ] 从品牌方接收设备+保修卡 → 分发给EPC
- [ ] 见证EPC与客户签署技术交底协议
- [ ] 见证EPC与客户签署质量安全免责协议
- [ ] 在三份文件上签字见证

**回传阶段：**
- [ ] EPC填写保修卡 → ND回收副本
- [ ] 检查保修卡: SN码、客户签名、EPC签名、日期是否完整
- [ ] 汇总保修卡副本 → 批量回传品牌方
- [ ] 保存一份副本备查

### 5.3 EPC操作指南

**安装前：**
- [ ] 确认已与ND签署MOU
- [ ] 确认品牌方已审核安装方案
- [ ] 准备IHY-50KH3S SOP要求的工具和材料
- [ ] 确认现场条件满足安装要求

**安装中：**
- [ ] 按SOP逐项施工（参照安装质量控制清单）
- [ ] 每完成一项在清单上打勾
- [ ] 关键节点拍照留档
- [ ] 与客户签署技术交底协议
- [ ] 与客户签署质量安全免责协议

**安装后：**
- [ ] 完成安装质量控制清单
- [ ] 配合品牌方完成调试
- [ ] 填写保修卡（安装日期、客户签字、EPC签字）
- [ ] 保修卡副本 → 提交ND
- [ ] 提交安装照片和测试记录

### 5.4 客户操作指南

**安装前：**
- [ ] 确认项目地址和安装条件
- [ ] 指派项目对接人
- [ ] 确认与EPC签署的技术交底协议和质量免责协议

**安装中：**
- [ ] 接受EPC的技术交底 → 确认理解后签字
- [ ] 确认已阅读免责条款 → 签字
- [ ] 验收安装质量 → 签字确认

**使用中：**
- [ ] 按操作手册使用设备
- [ ] 定期检查（参考SOP中的维护要求）
- [ ] 发现异常 → 联系EPC报修
- [ ] EPC不响应 → 联系ND升级
- [ ] ND不响应 → 直接联系品牌方

## 6. 异常处理与升级机制

### 6.1 报修升级路径

```
客户发现问题
    │
    ▼
第1级: EPC现场诊断 ─── 24小时内响应
    │ (EPC不响应)
    ▼
第2级: ND介入协调 ─── 48小时内响应
    │ (ND不响应)
    ▼
第3级: 品牌方介入 ─── 72小时内响应
    │
    ▼
质保范围内 → 品牌方安排维修/更换
非质保范围 → 报价维修
```

### 6.2 常见问题处理

| 问题类型 | 处理方式 | 责任方 | 时限 |
|----------|----------|--------|------|
| 设备不启动 | 远程诊断→现场排查 | EPC→品牌方 | 24h |
| 性能不达标 | 检查参数设置→系统调试 | 品牌方 | 48h |
| 安装缺陷 | EPC免费整改 | EPC | 按工期 |
| 客户误操作 | 远程指导→现场培训 | EPC | 24h |
| 通讯故障 | 检查接线→模块更换 | EPC | 24h |

## 8. 风险与应对

| 风险 | 影响 | 应对措施 |
|------|------|----------|
| EPC不配合签协议 | 质保无法激活 → 客户不满 | MOU中明确扣款条款 |
| 保修卡丢失/忘填 | 质保证据缺失 | 电子版（WhatsApp/邮件）并行 |
| ND执行力不足 | 文件回收率低 | 品牌方设专人周度跟进 |
| 客户不配合签字 | 流程卡在G1 | EPC提前沟通，明确签协议是安装前提 |
| EPC安装质量差 | 设备故障多 | 品牌方加强培训和资质审核 |
